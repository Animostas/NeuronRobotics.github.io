A Better DyIO

The USB DyIO, and the new WiFi DyIO, are robot controllers you access through Java or Python. These devices provide your apps high speed access to a wide, and reconfigurable, variety of inputs and outputs. These inputs and outputs are used to attach sensors and actuators commonly used to build robots. The DyIO also has real-time capabilities for dynamic control.

Bowler Studio

BowlerStudio is a robot development application that combines scripting and device management with powerful control and processing features.

In addition to the robots sensors and actuators, BowlerStudio brings the full power of OpenCV the image processing library. Built in is a kinematics engine based on D-H parameters. JCSG provides a fully featured CSG based cad engine. Groovy and Jython provide the scripting frameworks, with languages to come. Scripts can be loaded from files or from any GitHub Gist with the embeded WebKit browser. Gist and the browser based ACE editor provides Social Coding and Cloud integration, as well as full archived backups of your scripts.
